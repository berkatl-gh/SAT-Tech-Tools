# SAT Math Logic Solver (C++) 🎓💻

As a Software Engineering student preparing for the SAT, I developed this tool to bridge the gap between academic theory and practical programming. This project automates the solution steps for **Quadratic Equations**, a core topic in the SAT Math section.

### 🛠 Technical Approach
- **Language:** C++
- **Logic:** Implemented discriminant analysis ($\Delta = b^2 - 4ac$) to determine the nature of roots.
- **Edge Cases:** Handles $a=0$ errors and provides complex root analysis (relevant for advanced SAT topics).

### 🚀 Why This Matters
High-stakes testing like the SAT requires deep logical understanding. By coding these formulas in C++, I:
1. Solidified my understanding of quadratic functions.
2. Improved my C++ debugging and logic flow skills.
3. Created a reusable tool for my own study sessions.
